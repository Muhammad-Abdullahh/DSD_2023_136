1. Install Siemens Questa Advanced Simulator 2024.1 Win64

2. Copy folder "questasim64_2024.1" on top of original one
   (at setup by default C:\questasim64_2024.1 )

3. Copy folder "ProgramData" to C: with overwrite

4. Run "Questa_Sim_Licensing.reg" and confirm to add info into Windows Registry

5. Run "SolidSQUADLoaderEnabler.reg" and confirm to add info into Windows Registry

6. Reboot

7. Enjoy
